This plugin will add a configurable widget to display social media icons in your widget area(s). Currently supports adding: Twitter, Facebook, LinkedIn, Google+, Pinterest, RSS Feed, & YouTube.

== Description ==

This plugin will add a configurable widget to display social media icons in your widget area(s). Icons are 32x32, squared edges, and display inline.

Supported social media links include:

*   Twitter
*   Facebook
*   Google+
*   LinkedIn
*   YouTube
*   Pinterest
*   RSS

**Available Languages**

* English (en_US)
* Serbo-Croatian (sr_RS) - Borisa Djuraskovic at (http://www.webhostinghub.com/)

**UPGRADE NOTICE**

A change has been made to the social profile links if you are upgrading this plugin from a version prior to 1.0. The Social Media URLs are managed from a new area in the WordPress admin and will require you to enter your URLs again. 


== Installation ==

1. Upload 'simple-social-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your social media connections under Settings > Simple Social Widget. Add absolute path(s) to the desired social media sites. (If the input field is left blank, the icon will not appear on the website)

Using the Widget:
1. Activate and configure the Plugin
2. Navigate to your widgets control panel
3. Add the 'Custom - Simple Social Widget' to a widget area and add give it a title

Using the Shortcode:
1. Activate and configure the Plugin
2. Use the shortcode [simple-social-widget]
